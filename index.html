Paralegal

Explanation of Paralegal

We built a vector database to enable search across thousands of bylaws, reducing legal research time by 40% and improving search accuracy by 30%

How it Works

The User will send in a query which will get embedded into a vector that is compared to the thousands of Bylaws separated into chunks that we gathered into our database. We scraped the Toronto Bylaws website in order to collect all of the bylaws, and then embed them into vectors with over 380 dimensions to explain what information each chunk has. which we then compare the user's query to the database of bylaws and return the 7 most relevant Bylaws. That will then be sent to a Gemini API which will summarise what the relevant bylaws say compared to the user's query.

We won the best use of MongoDB because of our vector database.

Converting Images to black and White

Example of conversion

Project Overview

A technical exploration of low-level pixel manipulation, developed to understand how digital images are represented at the pixel level. The project focused on analyzing and visualizing raw pixel data using Python.


Key Details

  • Core Objective:

    Reverse-engineer pixel representation in digital images

  • Technical Approach:

    1. Loaded and parsed image data at the pixel level

    2. Decomposed each pixel into its RGB components

    3. Mapped grayscale values using the average of each RGB value

  • Tools Used:

    Python, Matplotlib, NumPy

Outcome

Demonstrated the mathematical relationship between color spaces, and Generated accurate grayscale representations using RGB values
