Converting HEIC doccuments to PNG

Project Overview

Developed a terminal-based file conversion system to transform HEIC images to PNG format. The primary challenge involved understanding user requirements through clear communication to deliver a simple, and effective solution.


Key Details

  • User Need:

    Batch conversion of HEIC files to PNG format with minimal complexity

  • Solution:

    Created a lightweight terminal tool requiring just one command execution

  • Implementation:

    Documented clear instructions for:

    1. Navigating to target folder

    2. Executing the conversion
      command

    3. Verifying output

Outcome

Delivered an efficient solution that eliminated manual file conversion, saving the user significant time while maintaining accessibility for non-technical users.

Converting Images to black and White

Example of conversion

Project Overview

A technical exploration of low-level pixel manipulation, developed to understand how digital images are represented at the pixel level. The project focused on analyzing and visualizing raw pixel data using Python.


Key Details

  • Core Objective:

    Reverse-engineer pixel representation in digital images

  • Technical Approach:

    1. Loaded and parsed image
      data at the pixel level

    2. Decomposed each pixel
      into its RGB components

    3. Mapped grayscale values
      using the average of each
      RGB value

  • Tools Used:

    Python, Matplotlib, NumPy

Outcome

Demonstrated the mathematical relationship between color spaces, and Generated accurate grayscale representations using RGB values
